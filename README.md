# KRGO - Full-Stack Business Automation & WhatsApp Bot

Welcome to **KRGO**, a professional full-stack solution for lead collection and automated client onboarding. This project features a React frontend, a Node.js/Express backend, and an integrated WhatsApp bot for seamless scheduling.

## 🚀 Quick Start

### 1. Prerequisites
- Node.js (v18+)
- MongoDB
- WhatsApp account (for bot)

### 2. Installation
```bash
# Install root dependencies
npm install

# Install backend dependencies
npm install --prefix backend
```

### 3. Environment Setup
Create a `.env` file in the `backend/` directory (see `.env.example`).
Create a `.env` file in the root directory (see `.env.example`).

### 4. Running the Project
```bash
# Start both frontend and backend in development mode
npm run dev
```
